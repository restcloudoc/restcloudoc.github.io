# Shell脚本

本节主要用于介绍 RestCloud 中相关 Shell脚本 的内容  
总结下来就是： `鸡肋`  
建议进行 命令行 相关操作时，通过 python脚本 进行  
或者仅仅用该脚本调用已存在的 Shell脚本  

# 脚本用法

因为该脚本仅仅可以调用一些简单命令，无法组装大型脚本  
如果需要组装大型脚本，可以使用 python 来进行 [shell命令组装](/scripts/python#shell命令组装)  

```bash
ls
&&
cd 
/home/test/mytest
&&
ls 
-al
```
